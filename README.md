## :keyboard: Запуск из терминала
Локальный запуск всех тестов:
```
gradle clean test -Denv=config/local
```

Удаленный запуск всех тестов:
```
clean
test
 -Dbrowser=${BROWSER}
 -DbrowserSize=${BROWSER_SIZE}
 -DworkingHost=${WORKINGHOST}
 ```
